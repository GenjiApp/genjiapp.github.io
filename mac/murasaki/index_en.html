Murasaki

The Scroll-based EPUB Reader for Mac OS X

  • Description

Description

Murasaki is a simple EPUB reader for Mac OS X (CANNOT open DRM-encrypted EPUBs).

Murasaki is a scroll-based reader like usual web browser, not a paginate-based reader like iBooks. Don't be trapped by dogma which is believing the page of books should be flipped.

By using this application, you can read EPUB books, view its navigation structure and metadata, search words, apply user style sheet, bookmark pages and so on.

If you cannot open sidebar, please refer to FAQ page.

Requirement
Mac OS X 10.6.6 or later

Features

Murasaki has various unique features.

Murasaki is developed for Mac OS X, by using Cocoa framework. Because Murasaki is a Cocoa application, Murasaki integrates with various useful features of Mac OS X.

Subwindow

In Murasaki ver. 1.6 or later, You can open a current page or a link in a subwindow by choosing menu item from Window or context menu.

Popover

In Murasaki ver. 1.6 or later and Mac OS X 10.7 Lion, You can open a link in a popover by Option-clicking a link element or choosing menu item from context menu.

Popover can be detached as a subwindow by dragging its frame.

Bookmark

You can add bookmarks, and can refer these bookmarked pages later.

To add bookmarks, click "Add Bookmark" button on toolbar or choose "Add Bookmark" menu from "Bookmarks" menu. To refer bookmarks, open "Bookmarks" pane at sidebar or "Bookmarks" menu from menu bar.

To remove bookmarks, click "Remove Bookmark" button on toolbar or choose "Remove Bookmark" menu from "Bookmarks" menu after open the bookmarked page.

Applying user style

If you don't like the styles of EPUBs, you can apply a user style sheet by using Murasaki. More information describes "Applying user style" in FAQ page.

Trackpad Gesture

Trackpad gestures are available on Murasaki. You can swipe to back/forward history, pinch to zoom in/out.

Popup Dictionary

Popup dictionary is appeared by pressing "Command + Control + D" keyboard shortcut when a mouse cursor is on a word.

Text to Speech

You can use "Text to Speech" that is buil-in feature in Mac OS X on Murasaki.

Includes Spotlight, Quick Look plug-ins

Murasaki ver. 1.4 (or later) includes Spotlight and Quick Look plug-ins for EPUB.

By installing Murasaki, you can Spotlight search EPUBs by their metadata (title, author and so on) and text contents, and can Quick Look preview EPUB's contents (only 10 HTML files per EPUB, 10 images per 1 HTML, cannot read external css files).

Vertical writing mode

If Safari 5.1 is installed, -epub-writing-mode (-webkit-writing-mode) property is available, then Murasaki can display vertical text.
