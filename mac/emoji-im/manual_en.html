Install

  1. Run “Emoji IM Installer.pkg”.
  2. Open System Preferences.
  3. Choose “Keyboard” Preferences.
  4. Choose “Input Sources” tab and click “+” button (and also check “Show Input menu in menu bar”).
  5. Choose “English” and “Emoji IM”, and click “Add”.
  6. Make sure “Emoji IM” is added.

Uninstall

  1. Open System Preferences.
  2. Choose “Keyboard” Preferences.
  3. Choose “Input Source” tab and “Emoji IM”, and click “-” button.
  4. Reboot system.
  5. Remove Emoji IM.app from /Library/Input Methods.

Usage

  1. Choose “Emoji IM” in Input menu.
  2. Type Emoji's name in UPPERCASE (type with Shift key or turn on Caps Lock key).
  3. Candidates window is shown.
  4. To select emoji, type Tab/Shift-Tab, Up arrow key/Down arrow key or Control-p/Control-n.
  5. To input a selected emoji, type Enter or double-click.
    • If you pressed Option key at this time, hexadecimal code point is inputted instead of emoji.

Emoji searching is ended when you type lowercase characters, hit Enter key or Escape key.

Caps Lock Mode

In Caps Lock Mode, you can only input Emojis when Caps Lock is on.

  1. Choose “Emoji IM” from Input menu.
  2. Choose “Preferences...” from Input menu.
  3. Check “Caps Lock Mode” in “General” tab.

Skin Tone

You can set skin tone.

  1. Choose “Emoji IM” from Input menu.
  2. Choose “Preferences...” from Input menu.
  3. Choose “Skin Tone” pop up button in “General” tab.

Ignore Eisu Key (for Apple JIS Keyboard)

If you use Apple JIS keyboard and you type Eisu key, a whitespace character is inputted unintentionally. To ignore Eisu key event:

  1. Choose “Emoji IM” from Input menu.
  2. Open “Preferences...” from Input menu.
  3. Check “Ignore Eisu Key” in “General” tab.

Register Favorites

You can register custom name of emojis:

  1. Choose “Emoji IM” from Input menu.
  2. Choose “Preferences...” from Input menu.
  3. Choose “Favorites” tab.
  4. Click “+” button.
  5. Input custom name to “Replace” text field and emoji to “With” text field.
  6. Click “OK” button.

Excluded Applications

You can register excluded apps to not enter emoji mode:

  1. Choose “Emoji IM” from Input menu.
  2. Choose “Preferences...” from Input menu.
  3. Choose “Excluded Apps” tab.
  4. Click “+” button.
  5. Choose applications you want to add to excluded apps, and click “Open” button.

Look up Emoji's name

To look up emoji's name:

  1. Choose “Show Character Viewer” from input menu.
  2. Look up emojis's name.
